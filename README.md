IcePick 0.4.0

Fast color palette workflow for Blender Texture Paint.

Features


[B]Open - IcePick
[Shift] + 1-8 - Select color slot
[Alt] + [W] - Next palette
[Alt] + [Shift] + [W] - Previous palette
[E] + [Mouse Drag] - Scrub through palette slots

Installation

1. Download IcePick.zip
2. Blender Edit Preferences  Add-ons
3. Install From Disk
4. Enable IcePick
5. Save Preferences

Hotkeys

 B  Open IcePick 
 Shift+1–8  Select Color Slot 
 Alt+W  Next Palette 
 Alt+Shift+W  Previous Palette 

Why?

I built IcePick because I wanted a faster way to switch painting colors while texture painting in Blender.

If it helps your workflow too, peace.
